Remote Monitoring Data API

API Reference

A REST API for managing remote monitoring projects and collected data.

Base URL https://capstone-2026-storage-server.dawson-packer.deno.net
Projects
GET /projects List all projects ▶

Returns an array of all projects.

Response
[
  {
    "id":   "d3b4a1c2-...",
    "name": "Togiak April 2026"
  }
]
POST /projects Create a project ▶

Creates a new project. Project names must be unique.

Request body
FieldTypeDescription
namestringUnique name for the project
Example
{
  "name": "Togiak April 2026"
}
Response 201 Created
{
  "id":   "d3b4a1c2-...",
  "name": "Togiak April 2026"
}
Records
GET /records/:id Get all records for a project ▶

Returns all records belonging to the specified project.

URL parameters
ParamTypeDescription
idstringThe project ID
Response
[
  {
    "id":          "9f2c3e1a-...",
    "projectId":   "d3b4a1c2-...",
    "temperature": -4.2,
    "latitude":    71.2906,
    "longitude":   -156.7886,
    "voltage":     3.71,
    "timestamp":   "2026-04-05T12:00:00Z"
  }
]
POST /records/:id Add a record to a project ▶

Adds a new sensor record to the specified project.

URL parameters
ParamTypeDescription
idstringThe project ID
Request body
FieldTypeDescription
temperaturefloatTemperature reading
latitudefloatLatitude coordinate
longitudefloatLongitude coordinate
voltagefloatVoltage reading
timestampstringISO 8601 datetime
Example
{
  "temperature": -4.2,
  "latitude":    71.2906,
  "longitude":   -156.7886,
  "voltage":     3.71,
  "timestamp":   "2026-04-05T12:00:00Z"
}
Response 201 Created
{
  "id":          "9f2c3e1a-...",
  "projectId":   "d3b4a1c2-...",
  "temperature": -4.2,
  "latitude":    71.2906,
  "longitude":   -156.7886,
  "voltage":     3.71,
  "timestamp":   "2026-04-05T12:00:00Z"
}
